Welcome to the Zesto Burgers restaurant website! This is a modern, responsive, and fully functional website for a premium fast food restaurant in Pakistan.

## 🎨 Customization

### Update WhatsApp Number
Replace `923001234567` with your WhatsApp number:
- In HTML: `https://wa.me/YOUR_NUMBER`
- In JavaScript: Same format

### Update Restaurant Details
- **Restaurant Name**: Change "ZESTO BURGERS" in the navbar
- **Location**: Update in the About section
- **Phone Number**: Modify in the Contact section
- **Working Hours**: Change in the Hero section

### Change Colors
Update CSS variables in `style.css`:
```css
:root {
    --primary-color: #dc2626;  /* Main red color */
    --dark-color: #1f2937;     /* Dark background */
    --light-color: #f3f4f6;    /* Light background */
}
```
